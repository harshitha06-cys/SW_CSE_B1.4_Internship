Position Property: dictates how an element is placed on a webpage. It controls whether an element stays in the normal page flow, or moves relative to its neighbours or locks it in a place where user scrolls. 1. Static: Normal page flow. (Default) 2. Relative: Moves away from its normal spot without disturbing neighbours 3. Absolute: Flies out of normalflow, positioning itself inside its closest relative parent 4. Fixed: Locks to the view port. Stays in the exact same spot on the screen when scrolling. 5. Sticky: Scrolls naturally with the page, then locks in place like FIXED when hits the edge.

Both display and position property control layout, the difference is that display creates the box and position moves the box.
